Overview

Survey of 100 learner experiences with the Physics Wallah platform. Focus areas include satisfaction, usability, content quality, value, recommendation likelihood, and open-ended feedback sentiment.

  • 100 structured responses (not publicly redistributed)
  • Anonymization pipeline drops likely PII patterns (email, phone, name, etc.)
  • Automated summary + figures produced via Python script

Workflow & Reproducibility

  1. Design form (Google Forms)
  2. Collect responses (private)
  3. Export Excel → local processing
  4. Run scripts/clean_and_analyze.py
  5. Generate artifacts/summary.md + figures
  6. Review & share aggregated insights only

Command (local): python scripts/clean_and_analyze.py

Data Usage & Privacy

Sensitive Educational Data. Not for redistribution. Treat all copies as confidential research material.
  • No direct identifiers should be committed (names, emails, phones).
  • Data license: CC BY-NC-ND 4.0 (non-commercial, no derivatives).
  • If a sensitive file leaks, open an issue tagged [PRIVACY].
  • Never attempt re-identification or merging with external datasets.

See full policy: DATA_POLICY.md

Key Components

  • clean_and_analyze.py: orchestrates anonymization + metrics.
  • Basic stats (counts, mean, std, quartiles).
  • NPS-style promoter/detractor classification (scaled if needed).
  • Optional sentiment (TextBlob) for open text, if installed.
  • Seaborn visual distributions & categorical counts.
