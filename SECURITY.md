# Security Policy

## Supported Versions
This project is a static site with no server code. Security issues are unlikely but may include:
- Malicious or unexpected data in uploaded/embedded files
- Mixed content or insecure links

## Reporting a Vulnerability
Please open a private security advisory on GitHub (Security > Advisory) or email the maintainer.

- Repository: https://github.com/Mandar123454/Physics-Wallah-Survey
- Preferred: GitHub private security advisory

Include:
- Description of the issue and steps to reproduce
- Browser and OS versions
- Screenshots or minimal dataset if possible

We aim to respond within 72 hours.
